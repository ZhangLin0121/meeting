# 🆘 微信小程序空白页面解决方案

## 🔍 问题概述

如果你的微信小程序打开后显示空白页面，这个指南将帮你快速诊断和解决问题。

## 📋 快速诊断步骤

### 1. 立即诊断 - 使用内置诊断工具

我们为你准备了一个专门的诊断页面：

1. **直接访问诊断页面**：
   - 在微信开发者工具中，手动输入页面路径：`pages/test/test`
   - 如果能打开，诊断工具会自动检测所有问题

2. **查看诊断结果**：
   - ✅ 应用状态：检查小程序是否正常初始化
   - 🌐 网络连接：验证网络状态
   - 💾 本地存储：检查用户登录信息
   - 🔐 登录状态：验证微信授权
   - 🔗 API连接：测试后端服务连接

### 2. 常见问题快速检查

#### 🔐 登录问题
**症状**：页面加载但没有内容，或显示登录相关错误

**解决方案**：
```javascript
// 检查控制台是否有这些日志
console.log('🔐 开始微信登录流程...');
console.log('✅ 微信登录完成');
```

**修复步骤**：
1. 清除小程序缓存：开发者工具 > 清缓存 > 全部清除
2. 重新授权：删除小程序后重新添加
3. 检查网络连接

#### 🌐 网络连接问题
**症状**：API请求失败，服务器连接超时

**检查方法**：
1. 打开开发者工具 > Network 面板
2. 查看是否有红色的请求失败
3. 检查请求地址是否正确：`https://www.cacophonyem.me/meeting`

**修复步骤**：
1. 确认网络连接正常
2. 尝试在浏览器访问：`https://www.cacophonyem.me/meeting/api/health`
3. 联系管理员检查服务器状态

#### 📱 微信版本问题
**症状**：特定功能无法使用，兼容性问题

**检查方法**：
```javascript
// 查看基础库版本
const systemInfo = wx.getSystemInfoSync();
console.log('微信版本:', systemInfo.version);
console.log('基础库版本:', systemInfo.SDKVersion);
```

**修复步骤**：
1. 更新微信到最新版本
2. 检查基础库版本是否支持（建议 >= 2.10.0）

#### 🏗️ 代码错误问题
**症状**：控制台显示JavaScript错误

**检查方法**：
1. 打开开发者工具 > Console 面板
2. 查看红色错误信息
3. 检查是否有语法错误或引用错误

## 🛠️ 详细排查流程

### 步骤一：基础检查
1. **开发环境确认**：
   - 使用微信开发者工具最新版本
   - 项目配置正确，AppID已设置
   - 基础设置中勾选"不校验合法域名"（开发阶段）

2. **网络设置**：
   - 确保网络连接正常
   - 如果使用代理，确保代理配置正确
   - 尝试切换网络环境

### 步骤二：登录流程检查
1. **查看登录日志**：
   ```javascript
   // 在控制台查找这些关键日志
   '🚀 小程序启动'
   '🔐 开始智能登录流程...'
   '✅ 微信登录完成'
   '📱 页面显示，开始初始化...'
   ```

2. **手动测试登录**：
   - 进入诊断页面
   - 点击"强制登录"按钮
   - 观察登录流程是否正常

### 步骤三：API连接测试
1. **健康检查**：
   ```bash
   # 在浏览器中测试
   https://www.cacophonyem.me/meeting/api/health
   ```

2. **会议室接口测试**：
   ```bash
   # 应该返回会议室列表
   https://www.cacophonyem.me/meeting/api/rooms
   ```

### 步骤四：存储和缓存清理
1. **清除小程序缓存**：
   - 开发者工具：清缓存 > 全部清除
   - 手机微信：发现 > 小程序 > 长按删除

2. **清除本地存储**：
   - 在诊断页面点击"清除数据"
   - 或在控制台执行：`wx.clearStorageSync()`

## 🔧 常用修复命令

### 开发者工具控制台
```javascript
// 查看当前用户信息
console.log('用户信息:', wx.getStorageSync('userInfo'));

// 查看应用全局数据
const app = getApp();
console.log('全局数据:', app.globalData);

// 手动触发登录
const WechatAuth = require('./utils/auth.js');
WechatAuth.performWechatLogin().then(result => {
    console.log('登录结果:', result);
});

// 测试API连接
const request = require('./utils/request.js');
request.get('/api/health').then(result => {
    console.log('API状态:', result);
});
```
